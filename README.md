# qiankun微前端

#### 介绍
qiankun微前端：

主应用采用vue2,子应用angular,子应用vue2,自用用react

#### 软件架构
软件架构说明


#### 安装教程
分别进入相应的目录

main  、 child-vue 、child-ng、child-react

1.  npm install
2.  npm run serve / npm start

#### 使用说明

1.  运行主应用 http://localhost:8080
2.  点击顶部菜单，显示对应的子应用
3.  子应用可独立进行访问， angular： http://localhost:4200
react:http://localhost:3000
